Стажировка Java Enterprise (Topjava)

Максимальный охват

Самые важные и необходимые Java разработчику темы, API, фреймворки, инструменты.

Структурированное изложение

Изложение материала нацелено на практическое использование и дополняется материалом для самостоятельной проработки. Темы идут последовательно, дополняя друг друга, для создания единого приложения.

Многомодульный maven. Многопоточность. XML. Веб сервисы. Удаленное взаимодействие (Masterjava)

Помощь с трудоустройством. Наши истории

В группе Slack выпускников происходит активное общение на все темы IT, обсуждение вступительных заданий, вакансии, цены на рынке труда, события, помощь, интересные видео и пр. Обновления нашей базы выпускников смотрят ~90 партнеров по трудоустройству (компании и индивидуальные рекрутеры).

Разработка Web приложения "База данных резюме" (Basejava)

Совместная разработка

Участники на протяжении всего курса работают с кодом проекта. Выполнение домашних заданий закрепляет полученные знания. В результате проект становится "Вашим".

Интервью в skype (голосом) на позицию Java разработчик.

Последовательная возрастание сложности

Сложность проекта возрастает постепенно, с кода "Hello!" в main() до финальной версии в Heroku. Слон съедается по кусочкам.

Общение, поддержка группы и преподавателя

По любым вопросам: git, трудоустройство, зарплаты программиста, java консультации, архитектура, новости IT и пр.

Доступность и материалов

Все видео доступно для просмотра онлайн и скачивания (в случае вебинаров - сразу после занятия). Записи разбиты по темам и тщательно отредактированы. Большое количество подобранных ресурсов для углубления в детали.

Оптимальное подобранное время

Занятия проходят раз в неделю, время необходимое для проработки пройденных тем и выполнения домашнего задания.

  • реализация сохранения в базы Postgres и HSQLDB на основе Spring JBDC, JPA(Hibernate) и Spring-Data-JPA
  • реализация и тестирование REST и AJAX контроллеров
  • реализация клиента на Bootstrap (css/js), datatables, jQuery + plugins.
