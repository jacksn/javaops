Разработка Web приложения
База данных резюме

  • используем: Java 8, IntelliJ IDEA, GitHib/Git, Сервлеты, JSP, JSTL, Tomcat, JUnit, PostgreSQL, GSON, JAXB
  • хранение резюме
    • в памяти на основе массива, отсортированного массива, списка и ассоциированного массива (Map)
    • в файловой системе (File API и Java 7 NIO File API)
      • в стандартной и кастомной сериализации Java
      • в формате JSON (Google Gson)
      • в формате XML (JAXB)
    • в реляционной базе PostgreSQL
  • деплой веб приложения
    • в контейнер сервлетов Tomcat
    • в облачный сервис Heroku
Веб приложение будет разрабатываться начиная со первого занятия, основываясь на базовых темах проекта: объектная модель, коллекции, система ввода-вывода, работа с файлами, сериализация, работа с XML, JSON, SQL, персистентность в базу данных (PostgreSQL), сервлеты, JSP/JSTL, веб-контейнер Tomcat, HTML, модульные тесты JUnit, java.util.Logging, система контроля версий Git.

Проект идет в индивидуальном режиме (старт в любое время) по готовым материалам с поддержкой в мессанджере Slack с личным куратором.
